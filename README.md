Project Title
RSVP Movies Global Release Strategy Analysis using SQL

Overview
This project uses SQL to analyze historical movie data for RSVP Movies, an Indian film production company planning to release a movie for a global audience. The analysis is based on movies released over the past three years and aims to generate data-driven insights to support strategic decision-making for international markets.

Business Problem
RSVP Movies has primarily catered to the Indian audience but plans to expand globally. To ensure success in international markets, the company wants to understand past movie performance trends, audience preferences, and factors contributing to successful films. The objective is to plan future projects analytically rather than relying on intuition.

Objective
Analyze historical movie data using SQL and provide actionable recommendations to help RSVP Movies make informed decisions about their global movie release strategy.

Data Description
The dataset consists of multiple relational tables containing information about movies, genres, ratings, production details, and release performance.
The data spans movies released over the last three years and is stored in a relational database created using a provided SQL script.

Approach
Loaded and explored the database using the provided SQL script
Analyzed data across multiple related tables using joins and subqueries
Wrote optimized SQL queries to answer business-driven questions
Used aggregations, filtering, and window functions where applicable
Validated results using checkpoints provided in the question script
Derived insights from each analysis segment and summarized findings

Key Insights
Certain genres consistently perform better across markets
Specific production factors strongly influence movie success
Audience ratings and vote counts provide clear signals of movie performance
Data-driven planning can reduce risk when entering global markets

Tools and Technologies
SQL
Relational Database Management System
Joins, Subqueries, Aggregations, Window Functions

Deliverables
SQL script file containing complete and optimized solutions to all questions
Executive summary in PDF format highlighting key insights and recommendations

Business Recommendations
Focus on high-performing genres with strong global appeal
Leverage historical performance metrics to guide production decisions
Use audience feedback and ratings as key indicators for future projects
